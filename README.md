🚀 Elite Tech Interview Operating System

📋 Professional Summary

The Elite Tech Interview Operating System is a unified, AI-powered preparation ecosystem designed to transition candidates from fragmented studying to a highly structured, data-driven mastery of technical interviews. By replacing disjointed spreadsheets and expensive coaching platforms, this dashboard serves as a personalized Staff-level mentor.

It combines advanced algorithmic telemetry with an RPG-style dependency skill tree to mathematically enforce learning and retention. Beyond just Data Structures and Algorithms (DSA), the platform features a complete suite of professional tools: a multi-language in-browser IDE, an AI-driven Behavioral (STAR) Architect, an ATS Resume Optimizer, and a Salary Negotiation Architect. Built with a sleek, responsive Glassmorphism UI, it empowers software engineers to conquer FAANG-level technical, behavioral, and architectural loops with total confidence.

✨ Core Features

🧠 Algorithmic Telemetry & Spaced Repetition (SRS)

Stop forgetting patterns. The dashboard utilizes the Ebbinghaus Forgetting Curve to track your performance. Rate your attempts (Struggled, Okay, Nailed It) and the system automatically dynamically generates a "Due for Review Today" queue to guarantee pattern retention until your interview day.

💻 Integrated Multi-Language Workspace

Say goodbye to context switching. A side-by-side workspace embeds the Monaco Editor (the engine behind VS Code) directly into the browser.

DSA: Run Python, C++, Java, and JavaScript instantly via the remote execution engine (Piston API).

SQL: Execute complex queries completely offline using the embedded WebAssembly SQLite engine (sql.js).

🐙 One-Click GitHub Portfolio Sync

Passively build your proof-of-work. Link your GitHub Personal Access Token, and seamlessly commit your mastered solutions directly from the in-browser IDE to a public repository to attract recruiters.

🌳 RPG-Style Skill Tree

Visualize your progress. Core concepts are mapped as nodes in a dependency skill tree. Master foundational algorithms like Arrays & Hashing to unlock advanced architectures like Graphs and 2-D Dynamic Programming.

🗣️ Behavioral (STAR) Story Architect

Input raw, unstructured bullet points about your past projects, and the embedded AI (Gemini) will restructure them into highly polished, impact-driven behavioral responses aligned perfectly with target frameworks like Amazon Leadership Principles or Google Googliness.

📝 AI Resume Intelligence (ATS Optimizer)

Upload your PDF resume or paste raw text. The built-in parser runs a deep-scan audit against Applicant Tracking System (ATS) algorithms, scoring your resume, identifying missing critical keywords, and rewriting weak bullet points into metric-driven statements.

💰 Offer Negotiation Architect

Maximize your total compensation. Input your base, equity, sign-on bonus, and competing leverage, and the AI coach will analyze the data to generate a highly professional, data-backed counter-offer email script.

📈 Predictive Readiness Heatmaps

Stop guessing when you are ready. The dashboard statistically maps your completed patterns against real-world interview probabilities for top tech companies (Meta, Google, Amazon, Apple, Microsoft) to give you an exact readiness percentage.

🛠️ Technical Architecture

This application is built for ultimate speed, portability, and zero-friction deployment.

Frontend: Pure HTML5, CSS3, and Vanilla JavaScript. (Zero build steps, Webpack, or Node modules required).

UI/UX: Custom-built Obsidian Glassmorphism design system with dynamic mathematical theme generation and deep Dark/Light mode integration.

Code Editor: Microsoft Monaco Editor (via CDN).

Local Execution: Pyodide and SQL.js (WebAssembly).

AI Engine: Google Gemini 2.5 Flash API.

Data Persistence: Firebase Auth & Firestore for real-time cross-device cloud syncing (with localStorage fallback and offline JSON import/export).

PDF Parsing: PDF.js (Mozilla) for client-side resume extraction.

🚀 Getting Started

Option 1: Web Browser (Instant Play)

Because the app is a self-contained .html file, installation is frictionless:

Clone or download this repository.

Double-click index.html to open it in any modern web browser (Chrome, Edge, Safari, Firefox).

(Optional) Add your Gemini API Key in the codebase (const apiKey = "YOUR_KEY_HERE";) to unlock the AI Mentor, STAR Architect, Resume Checker, and Offer Negotiator.

Option 2: Native Android App

You can wrap this dashboard into a standalone Android application that runs natively on your phone.

Please see the attached android_app_guide.md for step-by-step instructions on wrapping the HTML file in an Android Studio Web View with offline storage permissions.

🔒 Privacy & Data

Local First: By default, all progress, SRS data, and telemetry are stored securely in your browser's localStorage.

Cloud Sync: If you opt into the Firebase backend, your data syncs securely across devices.

Code Privacy: Code execution happens either entirely locally in your browser memory (WASM) or is streamed ephemerally to a sandboxed execution engine.
